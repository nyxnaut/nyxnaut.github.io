Freelancer's Calendar

The Google Calendar replacement you didn't know you needed (still w.i.p.)

This is my current project. I don't have the project page written yet so I have put the status and goals here. Its written in React, and uses a Node.js backend connected to a MongoDB instance. There are no class components, and a lot of hooks. Code viewable here, the live site is hosted here and an embed of the site may be found below.

Current Status

The site is functional! You log in through google, create and manage events, then view them in weekly or monthly views.

NOTE: I have a side goal of making my own web components for input, so I just put in a placeholder form for editing events. It assumes 24hr time and YYYY/MM/DD formatted dates.

Goals

  1. Reach feature parity with the core of Google Calendar

    Right now, that means allowing users to have multiple calendars, the ability to share events/calendars with each other and a few additional calendar views.

  2. Add features unique to the Freelancer's Calendar

    The most important feature enhancement is the ability to set flexible calendar items, and move them as a group. With these features, a meeting can run long without making you want to throw your whole calendar away for the rest of the day.

  3. Integration with other calendars (iCal/gCal)

    Event importing/exporting is a given, but I'd like the ability to keep the calendars in sync. Research into API's and license agreements is necessary before moving forward.
